# Previous Site Assets - 六神ノ間

## Hero Section
- Logo: 六神ノ間 with golden circular emblem featuring 6 oracle symbols
- Background: Dark mystical temple with blue/purple lighting and fog effects

## 6 Oracle Characters
1. **蒼真 (Souma)** - Blue hooded figure, time/clock icon, specialty: 運命の流れ・タイミング
2. **玲蘭 (Reira)** - Warm-toned woman with ornate headdress, heart icon, specialty: 癒し・恋愛・感情
3. **朔夜 (Sakuya)** - Purple-themed woman with tarot cards, grid icon, specialty: 数秘・性格・相性
4. **灯 (Akari)** - Light-themed figure, sun icon, specialty: タロット・恋愛・分岐
5. **結衣 (Yui)** - Dreamy figure, moon icon, specialty: 夢・無意識・直感
6. **玄 (Gen)** - Shield/guardian figure, shield icon, specialty: 守護・行動・現実的アドバイス

## Design Elements
- Color scheme: Deep purple/blue with gold accents
- Typography: Japanese serif for titles, clean sans-serif for body
- Card style: Dark cards with gradient borders
- Icons: Custom oracle symbols for each character
