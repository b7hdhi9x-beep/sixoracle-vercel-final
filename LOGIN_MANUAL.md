# 六神ノ間 - ログインマニュアル

## はじめに

六神ノ間では、**メールアドレスでの新規登録・ログイン**と**Manusアカウントでのログイン**の2つの方法をご利用いただけます。

---

## 1. 新規登録（メールアドレス）

### 手順

1. トップページの「**ログイン / 新規登録**」ボタンをクリック
2. ログインページが表示されたら「**新規登録**」タブをクリック
3. 以下の情報を入力：
   - **お名前**: ニックネームでも可
   - **メールアドレス**: 有効なメールアドレス
   - **パスワード**: 8文字以上
   - **パスワード（確認）**: 同じパスワードを再入力
4. 「**新規登録**」ボタンをクリック
5. 登録完了後、自動的にダッシュボードに移動します

### 注意事項

- パスワードは**8文字以上**で設定してください
- メールアドレスは1つにつき1アカウントのみ登録可能です
- 登録後すぐにサービスをご利用いただけます

---

## 2. ログイン（メールアドレス）

### 手順

1. トップページの「**ログイン / 新規登録**」ボタンをクリック
2. 「**ログイン**」タブが選択されていることを確認
3. 以下の情報を入力：
   - **メールアドレス**: 登録時に使用したメールアドレス
   - **パスワード**: 登録時に設定したパスワード
4. 「**ログイン**」ボタンをクリック
5. ログイン成功後、ダッシュボードに移動します

### ログインできない場合

- メールアドレスとパスワードが正しいか確認してください
- パスワードを忘れた場合は「**パスワードを忘れた方**」をクリックしてください

---

## 3. Manusアカウントでログイン

Manusアカウントをお持ちの方は、以下の手順でログインできます。

### 手順

1. トップページの「**ログイン / 新規登録**」ボタンをクリック
2. ログインページ下部の「**Manusアカウントでログイン**」ボタンをクリック
3. Manusのログイン画面が表示されます
4. お持ちのManusアカウントでログイン
5. ログイン成功後、自動的にダッシュボードに移動します

### 注意事項

- Manusアカウントをお持ちでない方は、メールアドレスでの新規登録をご利用ください
- 初回ログイン時に自動的にアカウントが作成されます

---

## 4. パスワードを忘れた場合

### 手順

1. ログインページで「**パスワードを忘れた方**」をクリック
2. 登録時のメールアドレスを入力
3. 「**リセットリンクを送信**」ボタンをクリック
4. 運営からパスワードリセットのご案内をお送りします

### 注意事項

- パスワードリセットは運営側で対応いたします
- リセット完了までお時間をいただく場合があります
- お急ぎの場合は、お問い合わせフォームよりご連絡ください

---

## 5. よくある質問

### Q: メールアドレスを変更したい

A: 現在、メールアドレスの変更機能は準備中です。お問い合わせフォームよりご連絡ください。

### Q: パスワードを変更したい

A: パスワードリセット機能をご利用ください。ログインページの「パスワードを忘れた方」から手続きできます。

### Q: アカウントを削除したい

A: お問い合わせフォームより、アカウント削除のご依頼をお送りください。

### Q: ログインしたのに占いができない

A: 無料プランの場合、累計5回までご利用いただけます。回数を使い切った場合は、プレミアムプランへのアップグレードをご検討ください。
