# 鑑定画面UI分析

## 現在の問題点
1. **メッセージの背景色が暗すぎる** - 濃い青/紫の背景に白文字で読みづらい
2. **フォントサイズが小さい** - 長文の鑑定結果が読みにくい
3. **余白が少ない** - テキストが詰まって見える
4. **テーブルが見づらい** - 背景色との区別がつきにくい
5. **見出しが目立たない** - セクション区切りが分かりにくい

## 改善案
1. メッセージバブルの背景色をより明るく/コントラストを上げる
2. フォントサイズを大きく（16px → 18px）
3. パディングを増やす（p-4 → p-6）
4. テーブルに明確な境界線を追加
5. 見出しに装飾を追加（アイコン、下線など）
6. 占い師のメッセージと自分のメッセージの区別を明確に
